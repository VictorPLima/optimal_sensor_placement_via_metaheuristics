# Metaheurísticas aplicadas ao problema do Posicionamento Ótimo de Sensores

Trabalho Final da disciplina de Tópicos Especiais em Ciência de Dados I do curso de Ciência da Computação da UFRJ.

Metaheurísticas implementadas: 
- Genetic Algorithm (GA)
- Differential Evolution (DE)
- Particle Swarm Optimization (PSO)

Também foi realizado de estudo de aplicação em cenário de otimização multiobjetivo onde preço de conjunto de sensores e cobertura de área conflitam.
